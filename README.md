# Esthetician's Sanctuary

A React + TypeScript + Vite website for Naji Beauty, showcasing services, gallery work, and booking flow for esthetician treatments.

## Repository

`https://github.com/smusoni/esthetician-s-sanctuary`

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Local Development

```sh
git clone https://github.com/smusoni/esthetician-s-sanctuary.git
cd esthetician-s-sanctuary
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - start local dev server
- `npm run build` - build production bundle
- `npm run preview` - preview production build
- `npm run lint` - run ESLint
- `npm run test` - run tests once
- `npm run test:watch` - run tests in watch mode
